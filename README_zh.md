# 通讯录管理系统

------
## 要实现什么？

    我们可以通过它实现对通讯录信息进行录入、显示、修改、删除、插入、排序、保存等操作的管理
    
## 功能有哪些？

    a.通讯信息录入
    b.通讯信息显示
    c.通讯信息保存
    d.通讯信息删除
    e.通讯信息修改
    f.通讯信息查询
        1.按编号查询
        2.按姓名查询
        3.按电话号码查询
    g. 退出系统
